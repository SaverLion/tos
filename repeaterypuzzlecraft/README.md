# Release Note.

## v1.0.0

### 修正内容

### 何をするもの？

マグナムオーパスを指定した回数、自動実行します。

### 使い方

0. 普段どおりマグナムオーパスUIでアイテムを配置したあと 調合ボタン  →開始する まで押します。

1. 数値入力ダイアログが表示されるので好きな回数(上限999)を入力します。

2. 確認ボタンを押すと調合が開始されます。

3. 終わるのを待ちます。

### どうすれば？

ipfファイルをインストールします。

### いつ動くの？

- マグナムオーパスの調合を開始する前
- マグナムオーパスの調合が終了した後
