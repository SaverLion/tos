# Release Note.

## v0.0.1

### 何をするもの？

任意のサイズのクイックスロットを任意の数だけ追加します。

### 使い方

- メニュー表示

フレーム（周りの枠らへん）を右クリック

Option

スロットサイズを変更できます

CreateNew

新しいスロットを追加できます

- スロットに登録したアイテムを使う

スロットを右クリック

- スロットからアイテムを削除

スロットをLALT＋左クリック

### どうすれば？

ipfファイルをインストールして、addonsフォルダに同名フォルダを用意します。
