## Latest Release Note.

### v1.1.0

#### ヘアカラー、称号追加

本バージョン以降に作成したデータは、ヘアカラーと称号を保存/復元できるようになります。

#### フレーム表示方法修正

インベントリを開くと追従して開かなくなりました。
閉じる場合は依然としてインベントリに追従します。

代わりに、インベントリアイコン右クリックメニューに[Open]が追加され、そこから開けます。
アドオンフレームが表示されるとインベントリが追従して開きます。
